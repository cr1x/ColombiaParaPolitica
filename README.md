### Prerequisites

> Install <a href="https://nodejs.org/en/" target="_blank">Node.js</a>

### Installation 🚀

1. Clone the repo
   ```sh
   git clone https://github.com/cr1x/ColombiaParaPolitica.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start project
   ```sh
   npm start
   ```
   :+1:

### Scripts

Build Demo (./public)

```sh
npm run build
```

Deploy Demo

```sh
npm run deploy
```

ReInstall project

```sh
npm run reinstall
```

Install package

```sh
npm install --save-dev name-package
```
